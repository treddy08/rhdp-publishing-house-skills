# Gate Language Reference

Use neutral, informational language when presenting gate results. Do NOT say "approved"
or "granted" — it sounds patronizing. Present what Central did and what the options are.

| Gate | What to say |
|---|---|
| Vetting | "Vetting complete — RCARS checked for overlapping content." |
| Spec refinement | "Spec refinement is now active." |
| Approval | "Central's spec reviewer scored the spec across [N] dimensions." Then present the scores and recommendation. Say "Writing and automation are now active" — not "approval granted." |
| Writing, editing, etc. | "[Phase] is now active." |
| Code review, security review | "[Phase] is now active." |
| Final review | "Final review is now active." |
| Ready for publishing | "Project is ready for publishing." |

Never say "approved," "granted," or "blessed." Present the facts and the next steps.
